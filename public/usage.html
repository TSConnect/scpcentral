Usage Guide

Your best place to find all things SCP!

Need information on an SCP?

Use the side bar on the left side to select the SCP you need information on. Our database is maintained by the community, meaning everyone can make edition.

Is the information accurate?

Every addition, deletion, or edition is moderated by the developers. Every single one of the operations stated needs to be proofread and approved by at least one developer to be published live. 

Can I create my own SCP?

This is planned to be an addition, but right now only confirmed SCP are allowed to be added.

How to read each SCP file

On each SCP file page, there is a section above and below a line. The section above is basic information about the SCP (Designation, Object Class), and the section below can be edited freely by the creators.

What about designations under another SCP?

If your scp designation is like SCPxxx-y, please include documentation on it under SCPxxx's file.
